# ha-energy-schema

Живая однолинейная схема (single-line diagram) энергосистемы для Home Assistant — проект VELMUR.

Сервис на **Go**, упакован как **Home Assistant add-on** (Docker) с **ingress**: запускается на той же машине, что и HA (HAOS VM), читает состояния заданных сущностей через Supervisor API и рендерит живой SVG-инфографику. HA отдаёт страницу своим origin, поэтому работает по HTTPS и в Android-приложении (без mixed-content).

## Что отображает
- Ввод №1 «Рыбхоз» — 3 однофазных стабилизатора (вход/выход/ступень/ток/пик по фазам), цвет линии по состоянию.
- Ввод №2 «Зелёный тариф» — напряжение/ток по фазам, состояние линии.
- Контактор (активный ввод).
- Солнце (PV), Инвертор (собств. потребление, усреднённое), АКБ (гейдж SOC + заряд/разряд).
- Генератор (состояние, наработка, до замены масла, подогрев ОЖ, V/нагрузка по фазам).
- Дом (нагрузка через инвертор / мимо инвертора).

## Структура
- `energy-schema/` — сам add-on (config.yaml, Dockerfile, Go-исходник).
- `repository.yaml` — метаданные репозитория аддонов (для добавления в HA как кастом-репозиторий, либо ставить как local add-on).

## Установка (local add-on)
Скопировать папку `energy-schema/` в `/addons/` на HAOS → Settings → Add-ons → Local add-ons → установить → включить, открыть из бокового меню «Схема».

## Конфигурация
Сущности задаются в опциях аддона (пока — захардкожены, далее вынесем в options).
